# qt-todoapp

# Run these commands to build.
### `NOTE:` These steps apply to qmake.

> git clone https://github.com/mhmdunl1/qt-todoapp.git

> cd qt-todoapp

> qmake TodoApp.pro

> make

> cd release 

> start TodoApp.exe
